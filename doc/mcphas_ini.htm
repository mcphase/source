McPhas.ini

 

This file controls the operation of the  module mcphas. It can be changed when the simulation is running and influences the output of the simulation.

It has 4 sections.

 

Section 1- XY-Phasediagram Paramters

 

Examples:

 

1)      Field (a direction) vs  Temperature: set xT=1 and yHa=1 (all others zero)

2)      Field (110 direction) vs Temperature: set xT=1 and yHa=1 and yHb=1 (all others zero)

3)      Field (a direction) vs Field (b direction) at constant temperature T=4 K: set xHa=1, yHb=1, T0=4 (all others zero)

 

Section 2 –Generation of Spin Configurations

 

  • Sets the range of the propagation vectors used to create initial sets of spin configurations. (using Miller indizizes h,k and l)
  • Size limit of magnetic unit cell: limits extension of magnetic unit cell and number of spins (useful to limit calculation time) – maxqperiod limits the number of primitive crystallographic unit cells, which are repeated in a given direction to generate the magnetic unit cell  (note that incommensurate structures are described approximately by large unit cells) – maxnofspins limits the number of spins in a magnetic unit cell
  • Monte Carlo Limits: nofrndtries sets the number of random changes of the initial spin configuration used in the mean field iteration process.

 

Section 3 – Parameters for Sub fecalc- Selfconsistency Process

 

  • Maxstamf: maximum standard deviation - limit to end the selfconsistency process                  

 standard deviation is defined by ...[bitte als formel] sta=sqrt(sum_{i=1}^{n} (newmf-old mf)i^2/n)                           

 the meanfield is given by mf=gj mb H [meV] (gj...lande factor, mb... bohr magneton)

  • maxnofmfloops: maximum number of selfconsistency loops, if no convergence is reached, meanfield iteration process is stopped
  • bigstep: mean field step ratio (=actual step/calculated step) to perform actually
  • maxspinchange: if  [als formel] sum_{i=1}^{n} abs(actual change of angular momentum <Ji> with respect to initial  configuration) > maxspinchange will  end selfconsistency process

 

Section 4 -  OUTPUT OF PHYSICAL PROPERTIES

Optional (all values can be zero)

 

  •  For thermal expansion and magnetostriction                  

nofspincorrs:  how many spinspin correlation functions   should be calculated

  • For Magnetic Diffraction (Neutrons, X-Rays): 

Maxnofhkls: calculation of  strongest reflections, this parameters defines the maximum number of reflection calculated at each point of the magnetic phase diagram

maxQ:  maximum scattering vector |Q|[1/A] of calculated hkl's
