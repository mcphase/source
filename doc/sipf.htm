Single Ion Property File

 

Used to describe magnetic single ion properties of rare earth ions.

 

Module type: the single ion properties can be described either by an anisotropic doublet (internal module kramers), by the full crystal field (internal module cfield) or by a user defined external module, which is loaded dynamically (linux only)

 

 

1.) Module cfield

The crystal field parameters are specific to the compound under consideration. Symmetry restricts the nonzero parameters – see e.g. Hutchings 1967

 

2.) Module Kramer

Instead of crystal field pararemeters the matrix  elements of the angular momentum for the ground state doublet |+-> have to be given. These are  |<+|Ja|->|, |<-|Jb|->|, |<+|Jc|->| and correspond to the saturation moments along a, b and c axis in Bohr magnetons divided by the Land’e factor.

[Formel:  M_sat=g_J \mu_B J (als gif. Evtl mit Roland Schedler rücksprache halten)]
